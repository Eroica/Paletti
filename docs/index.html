Paletti

Paletti is a small tool to quantize the colors of an image. This can be used to simplify the image visually or to extract certain key colors.

  • {% include icons/windows-brands.svg %} Download for Windows (.exe)
  • {% include icons/Gnomelogo-footprint.svg %} Download ELF binary (.AppImage)
  • {% include icons/android-brands.svg %} Download for Android β (.apk)
  1. Drag the slider (or scroll the mouse wheel) to increase/decrease the amount of colors. Less colors simplify the shapes of an image visually.
  2. Convert the image to black/white. This is useful to check the values of an image.
  3. Toggles always-on-top mode.
  4. The posterized image.
  5. Click on a color to copy its hex value into the clipboard.

Paletti is written in Kotlin and JavaFX for Windows. There is also an older version available which is written in Vala (using GTK), and a beta version for Android. The image processing is done using Leptonica.

The quantization process (Modified median cut quantization) usually extracts colors that are visually close to the original image, but it is not guaranteed to be exact. You can read more more about that here: Color quantization

If you like Paletti, you can also take a look at my other projects at GROUNDCTRL. Paletti’s icon is a 3D rendering of a katori buta, a Japanese figure usually made of ceramics. You can attach incense to a hook inside the body which is supposed to help repelling mosquitos.
