Paletti December 22, 2020

Paletti reduces an image to a desired amount of key colors. This is helpful if you need a color palette that matches a piece of artwork. You can copy the colors into your clipboard, or re-use the simplified image for something else.

I hope it is helpful for artists, designers, and other people working with colors.

You are welcome to raise issues or feature requests in Paletti’s GitHub repository.
